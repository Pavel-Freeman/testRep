# Основные части проекта Atlantm
  
>Основными частями проекта являются:  
- API
- CRON 
- Components
- ...
  
## CRON  
>Планировщик заданий Cron - это один из компонентов операционной системы Linux. Он используется для запуска на хостинге  определенных  скриптов  в нужное время, по расписанию.   
  
В **СRON** входят следующие PHP файлы:
- carexport.php
- cleandb.php
- clear_cache.php
- error_log_report.php
- import_catalog.php
- ... 
  
### import_catalog.php  
>Отвечает за импорт каталога товаров на сайт  
  
```PHP
<?php

$_SERVER["DOCUMENT_ROOT"] = empty($_SERVER["DOCUMENT_ROOT"]) ? __DIR__ . '/../../../' : $_SERVER["DOCUMENT_ROOT"];
define("CLI_ADMIN", true);
define("NOT_CHECK_PERMISSIONS", true);

require_once($_SERVER["DOCUMENT_ROOT"] . "/bitrix/modules/main/include/prolog_before.php");

use Atlant\Import\CCatalogImport;

CCatalogImport::getInstance()->run();

?>
```
#### Класс **CCatalogImport** 
>Класс отвечает за ...
  
**CCatalogImport** включает в себя следующие методы:
- getExecutePlan()
- initRating()
- initEngine()

#### **initEngine()**
>Метод отвечает за ...

Имеет следующую реализацию:
```PHP
function initEngine(){
        $list = new CEngineList();

        if($this->clearAll){
            $list->clearAll();
            return true;
        }

        $list->getAll();
        while ($item = $list->getNext()) {
            $this->engineList[$item->getID()]=$item->getXmlID();
        }

        return true;
    }
```
